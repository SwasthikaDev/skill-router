Find the right agent skill for any task — in one call.

Describe what you need in plain language. Skill-Router searches the live NANDA skills registry and returns the best-matching service plus a ready-to-run call — so an agent goes from “I need X” to an executable request in a single hop. No API key.

How it works

Three steps, fully autonomous — no human in the loop.

1

Describe the need

Send POST /find with a plain-language need. No auth, no setup.

2

Get a ranked match

Deterministic scoring over the live registry returns the best skills, each with why it matched.

3

Run the call plan

Each result includes a paste-ready curl and a link to that skill’s SKILL.md.

Built to be reliable

Live registry with a bundled snapshot fallback, so it answers even if upstream is down · deterministic (same need → same ranking) · self-correcting error messages · health check.
